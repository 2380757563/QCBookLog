### 5.2 关键数值业务含义

#### 5.2.1 阅读进度计算

**公式：**
```
阅读进度百分比 = (read_pages / page_count) * 100%
```

**示例：**
- page_count = 300
- read_pages = 150
- 阅读进度 = (150 / 300) * 100% = 50%

#### 5.2.2 节省金额计算

**公式：**
```
节省金额 = standard_price - purchase_price
折扣率 = (节省金额 / standard_price) * 100%
```

**示例：**
- standard_price = 39.00
- purchase_price = 35.50
- 节省金额 = 39.00 - 35.50 = 3.50
- 折扣率 = (3.50 / 39.00) * 100% ≈ 8.97%

#### 5.2.3 阅读时长统计

**单位转换：**
```
秒 → 分钟：total_minutes = total_reading_time / 60
秒 → 小时：total_hours = total_reading_time / 3600
秒 → 天：total_days = total_reading_time / 86400
```

**示例：**
- total_reading_time = 7200
- 总分钟数 = 7200 / 60 = 120 分钟
- 总小时数 = 7200 / 3600 = 2 小时

#### 5.2.4 平均阅读速度

**公式：**
```
平均阅读速度（页/分钟）= read_pages / (last_read_duration / 60)
平均阅读速度（页/小时）= read_pages / (last_read_duration / 3600)
```

**示例：**
- read_pages = 30
- last_read_duration = 1800（30分钟）
- 平均阅读速度 = 30 / (1800 / 60) = 1 页/分钟

### 6.2 性能优化

#### 6.2.1 查询优化建议

**当前查询问题：**
```sql
-- 低效查询：每次都进行子查询
SELECT b.*, 
  (SELECT name FROM authors a JOIN books_authors_link bal ON a.id = bal.author WHERE bal.book = b.id) as author,
  (SELECT val FROM identifiers i WHERE i.book = b.id AND i.type = 'isbn') as isbn
FROM books b;
```

**优化方案：**
```sql
-- 使用LEFT JOIN替代子查询
SELECT b.*, 
  GROUP_CONCAT(a.name, ' & ') as author,
  i.val as isbn
FROM books b
LEFT JOIN books_authors_link bal ON b.id = bal.book
LEFT JOIN authors a ON bal.author = a.id
LEFT JOIN identifiers i ON b.id = i.book AND i.type = 'isbn'
GROUP BY b.id;
```

## 字段定义
### 纸张类型 (paper1)
值 含义 0 未指定（默认） 1 胶版纸（双胶纸） 2 轻型纸 3 道林纸 4 铜版纸 5 牛皮纸 6 宣纸 7 进口特种纸

### 刷边位置 (edge1)
值 含义 0 无刷边（默认） 1 书口单侧 2 多侧（书口+天头/地脚） 3 全三边

### 刷边工艺 (edge2)
值 含义 0 无细分（默认） 1 基础单色 2 烫边（烫金/银） 3 磨边（毛边） 4 彩绘艺术刷边 5 鎏金高端刷边

## 🎯 功能特点
1. 两级分类设计 ：刷边采用 edge1（位置）+ edge2（工艺）的两级分类，与现有的 binding1+binding2 设计保持一致
2. 联动逻辑 ：当 edge1 选择"无刷边"时，edge2 控件自动禁用
3. 完整筛选 ：书籍列表页面支持按纸张类型、刷边位置和刷边工艺进行筛选
4. 向后兼容 ：新字段默认值为 0，不影响现有数据
5. 数据完整性 ：所有数据库操作都正确处理新字段，确保数据一致性
